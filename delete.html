Suppression des données

Supprimer les données locales

  1. ArchivEasy → Paramètres → Réinitialiser / Supprimer les données locales.
  2. Supprimez vos rappels/documents si nécessaire depuis l’interface.

Supprimer les sauvegardes Google Drive

  1. Dans ArchivEasy → Sauvegardes → Gérer → Supprimer la dernière sauvegarde.
  2. Ou via drive.google.com en supprimant le dossier “ArchivEasy Backup”.

Supprimer votre compte (si applicable)

Si une création de compte est proposée dans une future version, écrivez à support@archiveasy.com pour demander la suppression.
