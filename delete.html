Suppression des données

Vous gardez le contrôle total sur vos données.

Supprimer les données locales

  1. Ouvrez ArchivEasy → Paramètres → Réinitialiser / Supprimer les données locales.
  2. Supprimez vos rappels et documents individuellement si nécessaire depuis l'interface principale.

Supprimer les sauvegardes Google Drive

  1. Dans ArchivEasy → Sauvegardes → Gérer → Supprimer la dernière sauvegarde.
  2. Ou directement via drive.google.com en supprimant le dossier "ArchivEasy Backup".

Supprimer votre compte

ArchivEasy ne nécessite pas de création de compte. Si une fonctionnalité de compte est proposée dans une future version, vous pourrez demander la suppression complète de vos données en écrivant à contact@archiveasy.com.

Désinstallation

La désinstallation de l'application supprime automatiquement toutes les données locales stockées sur votre appareil. Les sauvegardes sur Google Drive doivent être supprimées manuellement si vous le souhaitez.

Besoin d'aide ?

Pour toute question concernant la suppression de vos données : contact@archiveasy.com
